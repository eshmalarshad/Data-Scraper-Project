# React + Node + Playwright SauceDemo Scraper

##  Overview

This project is a full-stack automation tool built using:

- React (Frontend)
- Node.js + Express (Backend)
- Playwright (Browser Automation)

It logs into the SauceDemo website using user credentials, scrapes product information from the authenticated dashboard, and displays it in a React UI.

---

##  Tech Stack

- Frontend: React, Axios, CSS
- Backend: Node.js, Express
- Automation: Playwright
